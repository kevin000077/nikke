# MarbleAim 分享版

当前版本：`v0.1.1`

## 使用

1. 先启动游戏并保持窗口化或无边框窗口模式。
2. 双击 `MarbleAim.exe`。
3. 第一次启动时，从列表中选择游戏窗口。
4. 后续会自动记住窗口标题；目标窗口不存在时会重新弹出选择列表。
5. 使用左上角“选择窗口”可以随时更换捕获目标。
6. 自动白框不准时，点击“设置白框”，依次确定左右竖线和上下横线。
7. 使用“退出助手”按钮结束后台进程。

运行期间会保留一个控制台窗口，里面显示配置位置、选中的目标窗口和启动异常。若图形界面没有出现，请保留控制台文字用于反馈。

程序不依赖固定安装路径，也不读取游戏目录。窗口匹配依据是可见窗口标题。

个人配置保存在：

```text
%LOCALAPPDATA%\MarbleAim\config.json
```

删除该文件可以恢复默认设置并重新选择窗口。

## 分辨率

识别和碰撞模型以游戏白色碰撞框宽度为基准缩放，支持 1080p、2K 和 4K。建议：

- Windows 显示缩放保持系统推荐值。
- 游戏不要跨越两个不同 DPI 的显示器。
- 调整游戏窗口尺寸后，等待助手自动重新识别或点击“重启识别”。

4K 会增加截屏和图像处理负载；完整方块识别仅在初次启动、换回合或手动重启时运行，瞄准阶段仍使用裁剪后的快速通道。

## 构建

双击 `build_release.bat`。生成文件位于：

```text
dist\MarbleAim.exe
```

这是单文件版本，首次启动会比文件夹版本稍慢，因为依赖需要解压到临时目录。
